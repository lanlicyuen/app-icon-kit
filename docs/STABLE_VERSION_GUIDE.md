## 🚀 使用方法

### **启动程序**
双击 `IconGenerator_Stable.exe`

### **操作步骤**
1. **选择图片**: 点击"选择图片"按钮（不支持拖拽）
2. **选择平台**: 勾选iOS、Android、Windows
3. **设置样式**: 调整圆角和边框（可选）
4. **选择输出**: 设置输出目录
5. **生成图标**: 点击"生成图标"
