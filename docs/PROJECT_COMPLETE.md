## 🚀 使用指南

### **快速开始**
1. **启动程序**: 双击 `IconGenerator_Pro.exe`
2. **选择图片**: 点击"选择图片"或拖拽图片到虚线框
3. **选择平台**: 勾选需要生成的平台（iOS/Android/Windows）
4. **设置样式**（可选）: 调整圆角和边框
5. **生成图标**: 点击"生成图标"按钮

### **Logo要求**
- **文件名**: `app_icon.png`
- **尺寸**: 1024x1024像素
- **格式**: PNG（支持透明背景）
- **位置**: `assets/app_icon.png`

### **输出文件**
每个平台生成完整图标集：
- **iOS**: 10种尺寸 (60pt-1024pt)
- **Android**: 6种尺寸 + XML配置
- **Windows**: 8种尺寸 + ICO文件
