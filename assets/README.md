# 应用图标资源文件夹

## 📁 文件夹结构

```
assets/
├── README.md                    # 说明文件
├── app_icon.png                 # 主应用图标 (1024x1024)
├── app_icon.ico                 # Windows图标文件
└── favicon.ico                  # 网站图标文件
```

## 📱 图标规格要求

### **主应用图标 (app_icon.png)**
- **文件名**: `app_icon.png`
- **尺寸**: 1024x1024 像素
- **格式**: PNG (支持透明背景)
- **用途**: 
  - 程序生成exe的图标
  - 所有平台图标的源文件
  - 应用商店展示图

### **Windows图标 (app_icon.ico)**
- **文件名**: `app_icon.ico`
- **尺寸**: 包含16x16到256x256多种尺寸
- **格式**: ICO格式
- **用途**: Windows桌面应用图标

### **网站图标 (favicon.ico)**
- **文件名**: `favicon.ico`
- **尺寸**: 16x16, 32x32, 48x48
- **格式**: ICO格式
- **用途**: 网站浏览器图标

## 🎯 使用方法

1. **准备图标**: 将您的1024x1024像素PNG图标重命名为 `app_icon.png`
2. **放置文件**: 将 `app_icon.png` 放入 `assets/` 文件夹
3. **构建exe**: 运行构建脚本时会自动使用此图标

## ⚠️ 注意事项

- 确保图标为正方形 (1024x1024)
- 建议使用简洁的设计，避免复杂细节
- 支持透明背景的PNG格式最佳
- 图标内容建议避免包含文字

## 🔧 自动生成

如果您只有 `app_icon.png`，程序会自动生成：
- Windows ICO文件
- 各种尺寸的PNG图标
- 适配不同平台的图标集
